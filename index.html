Lista de Funciones en Javascript

  • Function Flecha o por Expresion

    • Esta funcion te permite declarar con variables como const y let y expresar con un signo de flecha y todo en una sola fila.

    Ejemplo :

    const suma=(a, b) => a + b;

    const suma=(a, b) => a + b; return a * b;

  • Function Hoisting

    • Esta funcion es como se elevan las funciones a alo nivel para ejecutarse.

    Ejemplo :
    console.log(suma(5, 6));
    console.log(multiplicar(5, 6));

    const suma=(a, b) => a + b;

    const multiplicar=(a, b) => a * b; return a * b;

  • Function Parametros por Defecto

    • Esta funcion asume por defecto los parametros que se le pasa en el argumento o por defecto imprime UNDEFINED.

    Ejemplo :
    const saludar=(name='sin nombre') => 'hola mi nombre es ${name}';
    console.log(saludar());

    3 puntos suspensivos seguido de la palabra rest va asumir los valores
    cuando llamemos a ala funcion.

    Ejemplo :

    function lista(name, lastName, ...rest) ;
    console0.log(name);
    console0.log(lastName);
    console0.log(rest);

    • Esta funcion te permite declarar con variables como const y let
    La diferencia es que la variable const no varia, mientras que la
